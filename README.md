## 🎯 Задача проекта

**Ваша цель** — найти и исправить все баги в файле [`styles/output.css`](styles/output.css:1), чтобы стили из [`output.html`](output.html:1) совпадали с эталонным изображением.

### Инструменты для отладки:

- **Слайдер прозрачности** — накладывайте CSS-рисунок на эталон для точного сравнения
- **🔍 Режим различий** — визуально выделяет несовпадающие области
- **📸 Сравнить с эталоном** — точный анализ совпадения в процентах

## 🚀 Быстрый старт

### Требования

- [Node.js](https://nodejs.org/) версии 14.0 или выше

### Установка зависимостей

```bash
npm ci
```

### Запуск проекта

```bash
npm start
```

После запуска приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)
